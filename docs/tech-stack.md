# Tech Stack

- **Framework:** [React Router v7](https://reactrouter.com/llms.txt), [React 19](https://react.dev/llms.txt), [TypeScript](https://www.typescriptlang.org)
- **Bundler:** [Vite](https://vite.dev/llms.txt)
- **UI:** [shadcn/ui](https://ui.shadcn.com/llms.txt), [Tailwind CSS v4](https://tailwindcss.com/llms.txt), [Lucide](https://lucide.dev), [Motion](https://motion.dev/docs/react)
- **State & persistence:** [Legend State v3](https://legendapp.com/open-source/state/v3/llms.txt)
- **Data table:** [TanStack Table](https://tanstack.com/table/latest/docs/llms.txt)
- **Forms:** [React Hook Form](https://react-hook-form.com/docs)
- **Charts:** [Recharts](https://recharts.org/docs/en-US)
- **Date & time:** [date-fns](https://date-fns.org)
- **Backend & database:** [Supabase](https://supabase.com/llms.txt) (Postgres, Auth, Row Level Security)
- **Authentication:** Supabase Auth — Google OAuth
- **ID generation:** [uuid](https://github.com/uuidjs/uuid) — wrapped in `generateId()` from `~/lib/utils`
- **Package manager:** [npm](https://npmjs.com)
- **Deployment:** Vercel (static SPA)
